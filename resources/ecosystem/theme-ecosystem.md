### Theme Architecture

**Theme Header**
```css
/*
Theme Name: Example Theme
Theme URI: https://example.com/theme
Description: Theme description
Author: Author Name
Author URI: https://example.com
Version: 1.0.0
License: GPL v2 or later
License URI: https://www.gnu.org/licenses/gpl-2.0.html
Text Domain: example-theme
Domain Path: /languages
*/
```

**Theme Structure**
```
theme-name/
├── style.css
├── index.php
├── functions.php
├── screenshot.png
├── readme.txt
├── header.php
├── footer.php
├── sidebar.php
├── single.php
├── page.php
├── archive.php
├── search.php
├── 404.php
├── assets/
│   ├── css/
│   ├── js/
│   └── images/
├── inc/
├── template-parts/
├── languages/
└── templates/
```
